# Examples

This directory contains practical examples of using PivotUI components in real-world scenarios.

## Examples Structure

- **`basic/`** - Simple component usage examples
- **`forms/`** - Complex form implementations
- **`dashboards/`** - Business dashboard examples
- **`data-tables/`** - Advanced data table implementations
- **`themes/`** - Custom theme examples
- **`integration/`** - Framework integration examples

## Running Examples

Each example can be run independently:

```bash
cd examples/basic
npm install
npm run dev
```
