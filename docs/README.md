# Documentation

This directory contains comprehensive documentation for the PivotUI design system.

## Structure

- **`getting-started/`** - Installation and setup guides
- **`components/`** - Individual component documentation
- **`design-tokens/`** - Design token documentation
- **`examples/`** - Usage examples and patterns
- **`migration/`** - Migration guides between versions
- **`contributing/`** - Contribution guidelines

## Documentation Format

Each component should have:
1. **Overview** - Purpose and use cases
2. **Props/API** - Complete API documentation
3. **Examples** - Code examples with live demos
4. **Accessibility** - ARIA guidelines and keyboard navigation
5. **Design Guidelines** - Visual design principles
6. **Best Practices** - Usage recommendations

## Building Documentation

```bash
npm run docs:build
npm run docs:serve
```
